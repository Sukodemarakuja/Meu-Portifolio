## 💾 Sobre o Sistema

Seja bem-vindo ao **Suko_System**. Este não é apenas um portfólio estático comum; é uma aplicação web completa desenvolvida em **Python + Flask** estruturada para funcionar como um sistema de abas interativo com estética retrô/cyberpunk. 

O objetivo deste projeto é centralizar e expor minhas automações, ferramentas de visão computacional e projetos backend, servindo como porta de entrada para a minha primeira oportunidade real no mercado de desenvolvimento de software.

---

## 🛠️ Stack Tecnológica & Arquitetura

O sistema foi construído pensando em boas práticas de backend, isolamento de dados e robustez:

* **Backend:** Python 3.14 + Flask (Gerenciamento de rotas e renderização dinâmica).
* **Frontend:** HTML5 customizado + CSS Grid Layout (Interface em matriz responsiva 2x2 sem frameworks pesados).
* **Banco de Dados:** Estruturado localmente em formato `JSON`, permitindo escalabilidade para novos projetos.
* **Segurança (Error Handling):** Sistema blindado com blocos `try/except` para garantir 100% de uptime caso haja falha na leitura dos arquivos de dados.
* **Ambiente:** Gerenciado de forma ultra-rápida utilizando o ecossistema `UV`.

---

## 📂 Projetos Cadastrados no Sistema

Atualmente, o banco de dados do sistema renderiza dinamicamente ferramentas como:
1. **Manga Translator:** Script automatizado para Visão Computacional, integração de tecnologia OCR (Pytesseract) para extração de textos, tradução em lote e manipulação digital de imagens.
2. **Projeto P:** Automação integrada com APIs de Inteligência Artificial para manipulação de arquivos e processamento de dados.
3. **Suko System (Este Portfólio):** Aplicação Flask integrada que você está navegando agora.
